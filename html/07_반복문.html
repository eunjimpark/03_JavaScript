for 문

    - 프로그래밍의 흐름을 제어하는 제어문 중 하나로 특정 코드를  반복할때 사용

    -끝이 정해져 있는 반복에서 주로 사용
    ex)몇 번만 반복

    [작성법]
    for([1]초기식; [2]조건식; [3]증강식){
        조건식이 true인 경우 반복 수행할 코드;
     }

     //1~4까지 1회반복후 , 2~4를 계속 반복 
     //조건식이 false가 되면 바로 종료

     초기식: for 문을 제어하는 용도의 변수 선언

     조건식: for문의 반복 여부를 지정 식
            -true일때 반복, false일때 반복 종료
            -보통 초기식에 사용된 변수를 이용해서 조건식을 작성

     증강식: 초기식에 사용된 값을 변환 시키는 식

//1부터 10까지 1씩 증가하며 출력
//1부터 20까지 1씩 증가하며 출력
//5부터 15까지 1씩 증가하며 출력
//1부터 30까지 2씩 증가하며 출력
//1부터 10까지 모든 정수의 합

      while문

          -끝이 정확하게 정해져 있지 않은 반복에서 주로 사용
    (ex : 어떤 값이 입력 되기 전 까지 반복)

    [작성법]
    while(조건식){
        조건식이 true 인 경우 반복수행할 코드;
    }

    -조건식이 true인 경우 내부 코드를 1회 반복후 다시 조건식 검사 진행
    ->flase가 나올때 까지 무한 반복

      메뉴 주문하기

      1. 김밥 : 3000
      2. 라면 : 3500
      3. 돈까스 : 5000
